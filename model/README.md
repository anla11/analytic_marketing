# Base Processing: Common CS-level functions

## Overview

```
├── README.md                      <- The top-level README for developers using this project.
│
├── analytics                      
│   └── detector				   <- e.g. feature impacts
│   └── <feature_extraction>	   <- domain-based features extraction: purchase processing, survival analysis          
│
├── feature_processing             <- e.g. pca, feature scoring
│
├── predicting                     <- regression or classification with Pytorch/Pyro
│
├── clustering                     <- extract clusters and visualization with sklearn
│
├── post_processing                <- e.g. get threshold or binarize
│
├── evaluating                     <- get list of values by evaluation metrics
│
├── requirements.txt               <- Make this project installable with `pip install -r`
│
├── setup.py                       <- Make this project installable with `pip install -e`
```
